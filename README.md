## Real-Time Energy Consumption Prediction System
## 📌 Overview

This project is a Machine Learning-based Energy Consumption Prediction System developed to estimate building energy usage using environmental and occupancy-related factors. The system predicts energy consumption based on inputs such as building size, number of occupants, appliances used, temperature, building type, and day category.

The project combines Data Analysis, Feature Engineering, Machine Learning, and Streamlit Dashboard Deployment into a complete end-to-end AI application.
